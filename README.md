####bbs

**1.创建一个maven项目**
* 加载包
  * javax.servlet-api
  * json-lib
  * mysql-connector-java
  * jsp-api
  * commons-beanutils
* 创建资源包和Java包
  * 资源包：resource
  * Java包：java

**2.开始项目**
* 在Java包下分层
  * dao（数据层）
  * entity（实体类层）
  * service（业务逻辑层）
  * servlet（控制层）
  * untils（工具类层）
* 在webapp下建试图
  * js------存放js文件
  * css----存放css文件
  * web---存放页面
